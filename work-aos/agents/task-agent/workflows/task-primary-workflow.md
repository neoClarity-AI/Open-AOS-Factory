# Task Primary Workflow

Primary workflow for the Task / Commitment Agent.

1. Capture a task with priority, due date, and source.
2. Detect overdue and at-risk commitments.
3. Surface items in the daily brief and weekly review.
4. Log significant prioritization or commitment changes.
