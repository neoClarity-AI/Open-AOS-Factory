# Review Primary Workflow

Primary workflow for the Review / Reflection Agent.

1. Select the cadence: weekly-light, monthly-deep, or quarterly-strategic.
2. Run the matching review workflow and gather findings.
3. Audit completeness, consistency, and staleness across the AOS.
4. Refresh the user guide on the monthly cadence.
5. Propose improvements; apply file changes only after `Proceed`.
