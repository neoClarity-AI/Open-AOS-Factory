# Project Manager Primary Workflow

Primary workflow for the Project Manager Agent.

1. Capture a project brief using the project-brief template.
2. Scaffold the project folder (status, decisions, assets, archive).
3. Set milestones, risks, and stakeholders.
4. Update `/memory/projects.md` with lifecycle state.
5. Record kickoff and key changes in the project's decisions file.
