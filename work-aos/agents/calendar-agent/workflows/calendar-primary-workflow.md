# Calendar Primary Workflow

Primary workflow for the Calendar / Scheduling Agent.

1. Read the calendar and surface today's commitments and focus blocks.
2. Detect conflicts and over-commitment.
3. Propose scheduling options.
4. Prepare any calendar change as a Level 2 action and wait for `Proceed`.
5. Log significant scheduling decisions.
