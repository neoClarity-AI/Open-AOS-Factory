# End-of-Day Shutdown Workflow

Owned (user-facing) by the Chief of Staff Agent.

1. Confirm what was completed vs. carried over.
2. Task Agent updates statuses and rolls over open items.
3. Capture any decisions (route to decision-capture workflow).
4. Memory Agent captures durable items surfaced today.
5. Note tomorrow's top priorities.
