# Memory Primary Workflow

Primary workflow for the Memory Agent.

1. Receive a memory candidate.
2. Test memory-worthiness; discard if not durable/useful/relevant.
3. Classify sensitivity — route sensitive items through Security approval.
4. Route to the right file, set a review date, and log the routing decision.
