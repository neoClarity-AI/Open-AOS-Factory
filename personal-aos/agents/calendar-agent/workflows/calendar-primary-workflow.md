# Calendar Primary Workflow

Primary workflow for the Calendar / Scheduling Agent.

1. Review the calendar within scope.
2. Detect conflicts and openings.
3. Propose changes and focus blocks.
4. Apply changes only after `Proceed` (especially events involving others). Log approved changes.
