# Project Manager Primary Workflow

Primary workflow for the Project Manager Agent.

1. Kick off via the project-kickoff workflow and project-brief template.
2. Plan milestones, risks, and stakeholders.
3. Post status updates and run project reviews.
4. Archive on approval; log structural conventions and significant decisions.
