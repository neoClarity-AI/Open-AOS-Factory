# Weekly Review Workflow

Primary owner: Review / Reflection Agent. Cadence: weekly-light.

Guiding question: **"What needs follow-up soon?"**

1. Review open tasks, at-risk commitments, and waiting items.
2. Check project momentum and loose ends.
3. Light memory pass (defer deep hygiene to monthly).
4. Capture findings; propose improvements (file changes gated by `Proceed`).
