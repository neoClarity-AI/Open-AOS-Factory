# Project Kickoff Workflow

Primary owner: Project Manager Agent.

1. Capture the project brief using `/templates/project-brief-template.md`.
2. Create the project folder with standard files plus `/assets` and `/archive`.
3. Set initial milestones, risks, and stakeholders.
4. Reflect the new project in `/memory/projects.md`.
5. Record kickoff in the project's `project-decisions.md`.
