# Quarterly Review Workflow

Primary owner: Review / Reflection Agent. Cadence: quarterly-strategic.

Guiding question: **"Is the whole system still aimed at the right goals?"**

1. Revisit purpose and outcomes that matter most.
2. Assess whether installed agents still match real needs; recommend adding/pausing agents.
3. Review permission posture and tool access with the Security Agent.
4. Capture strategic decisions and a refreshed plan.
