# Memory Review Workflow

Primary owner: Memory Agent, supported by the Review / Reflection Agent.

1. **Weekly-light:** scan recent entries for accuracy and correct routing.
2. **Monthly-deep:** check for stale, duplicated, or misplaced entries.
3. Mark stale entries and supersede outdated facts — never silently delete.
4. Archive-with-approval where removal is warranted.
5. Log hygiene actions.
